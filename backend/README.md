# Проект Mesto бэкенд

## Стэк:
* Express.js
* Mongodb
* Rest API

## Директории

`/routes` — папка с файлами роутера
`/controllers` - папка с файлами контролерров
`/models` -  папка с файлами схем моделей

## Запуск проекта

`npm run start` — запускает сервер   
`npm run dev` — запускает сервер с hot-reload
